# Schema, vecka 20
###### Backend med Node.js, vecka 1 av 5

## Introduktion

För att bygga moderna webbapplikationer behöver vi en stabil och flexibel backend. I denna modul får du lära dig grunderna i Node.js och Express – från att skriva dina första serverrader till att bygga strukturerade REST API:er. Vi introducerar även middlewares och hur de används för att hantera exempelvis loggning, fel och inkommande requests på ett effektivt sätt. Fokus ligger på att förstå hur backend och frontend kommunicerar med varandra. När du är klar med denna modul kommer du kunna skapa egna API:er med tydlig struktur och smart requesthantering!

## Mål för veckan:

1. Kunna bygga en enkel server med Node.js och Express
2. Förstå hur moduler importeras och används (core, externa och egna)
3. Kunna skapa ett REST API med flera endpoints
4. Förstå hur JSON används som dataformat vid HTTP-kommunikation
5. Kunna skapa och använda middlewares för att strukturera applikationen
